# telegram.demo

Testing telegram bot with the library [pengrad/java-telegram-bot-api](https://github.com/pengrad/java-telegram-bot-api)

## set token

Set bot token on from @BotFather on `src/main/resources/application.properties`

```
bot.token=bot_token_value_from_bot_father
```

## run

`mvn spring-boot:run`
